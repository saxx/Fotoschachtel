# Title
Fotoschachtel


# Short description
Alle Fotos eines Events blitzschnell mit den anderen Gästen teilen.


# Keywords
Foto Fotos teilen Hochzeit Party Event


# Description
Wer war noch nicht auf einer Party, einer Hochzeit, einem Gruppenausflug oder sonst irgendeinem Event, wo es tags drauf wahnsinnig kompliziert war, mit allen Gästen Fotos auszutauschen?

Fotoschachtel löst dieses Problem, indem es jedem Gast eines Events ermöglicht, blitzschnell Fotos zu machen und live mit allen anderen Gäste zu teilen.

Keine endlosen E-Mail-Kettenbriefe mehr. Keine Social Media Sites wo doch wieder nicht alle Gäste erreichbar sind. Kein Hantieren mehr mit Dropbox-Links oder Einweg-Kameras aus den 90ern. Fotoschachtel macht Fotos machen und teilen so einfach wie möglich:

1. Smartphone rausholen
2. Fotoschachtel starten
3. Fotos machen
4. Fertig :)
